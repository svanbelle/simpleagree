 This shiny application determines different agreement indexes for binary scale when two or more raters are involved. This is possible in the frequentist and the Bayesian frameworks. 
 
 To use this app from GitHub, copy paste the following code in R
 
library(shiny)
runGitHub(repo="simpleagree",username="svanbelle",ref="main")

Note that you should have the package "shiny" installed.
